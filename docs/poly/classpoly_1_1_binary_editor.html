Detailed Description

template<class Real>
class poly::BinaryEditor< Real >

Interface that describe an entity that can modify the structure and the content of a binary and can retrieve some info about its structure. A class that wants to implement this interface must inherit from its own template specialization:

class Impl : public BinaryEditor<Impl> { ... };
