Detailed Description

template<std::size_t bytes, std::size_t check = bytes, typename = impl::range<>>
struct poly::BlockBuilder< bytes, check, typename >

The BlockBuilder is an interface for the generations of new Block object from arrays of bytes. The size of the Block to construct has to be specified by the non-type template parameter bytes.
