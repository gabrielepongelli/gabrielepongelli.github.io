Detailed Description

template<class Real, class Cipher, class Compiler, class Editor>
class poly::PolymorphicEngineBase< Real, Cipher, Compiler, Editor >

Abstract class that implement some methods common for all the OS. It requires to the concrete class that extends it to implement the following protected member methods:

  • void generate_syscall(Address, std::size_t, asmjit::Label &)
