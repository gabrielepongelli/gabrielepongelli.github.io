Detailed Description

template<EncryptionAlgorithmType E = EncryptionAlgorithmType::kNone>
struct poly::EncryptionAlgorithm< E >

An EncryptionAlgorithm implement the algorithm to use for the encryption and the decryption. The right algorithm to use must be specified through the non-type template parameter E, which defaults to kNone.
